# FE Workspace Deployment Checklist

## 1) Pre-Deployment Readiness
- [ ] Workspace access and entitlements confirmed for FE team.
- [ ] Environment topology approved (`dev`, `test`, `prod`).
- [ ] Source integrations and credentials validated.
- [ ] Security and governance baseline controls approved.
- [ ] Release window and rollback window approved.

## 2) Data Platform Deployment
- [ ] Ingestion pipelines deployed and smoke-tested.
- [ ] Transformation jobs deployed and run successfully.
- [ ] Data quality controls enabled and alert routing verified.
- [ ] Freshness SLA monitors active and threshold-configured.

## 3) Governance and Security Validation
- [ ] Role-based access checks passed.
- [ ] Sensitive data controls validated.
- [ ] Lineage visible for priority metrics and reports.
- [ ] Audit evidence capture validated.

## 4) Reporting and Consumer Validation
- [ ] Dashboard datasets connected and refreshed.
- [ ] Dashboard UAT pass confirmation recorded.
- [ ] Genie space tested with approved question set.
- [ ] Subscription and publication settings validated.

## 5) Operational Handover Readiness
- [ ] On-call and escalation contacts confirmed.
- [ ] Incident runbook and replay process tested.
- [ ] Support SLAs agreed and documented.
- [ ] Handover runbook walkthrough completed.

## 6) Go-live Approval
- [ ] Business owner sign-off.
- [ ] Platform owner sign-off.
- [ ] Governance/control sign-off.
- [ ] FE delivery sign-off.
